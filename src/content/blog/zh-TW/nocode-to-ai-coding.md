我離開工程師這個身份，已經 14 年了。

中間做過 PM、數據分析師，偶爾寫點 Python 腳本。但要自己從零寫一個完整產品？用傳統的方式，門檻太高。

所以我去找了 No-Code。

## No-Code 的嘗試

2024 年，我遇到 No-Code。第一個學的是 Bubble。

「不用寫程式也能做產品？」

我花了半年研究 Bubble。動態資料、Plugin 串接、複雜邏輯，都摸透了。還用它做出了一個 SaaS 產品。

我寫了 8 篇 Bubble 教學文章。用中文搜尋「Bubble.io」，我的文章排 Google 第二。第一是 Bubble 官網。

以為找到答案了。

但問題來了。

產品做出來了，Landing Page 呢？Bubble 的 SEO 很差，速度也慢，不適合。Chrome Extension 呢？Bubble 不支援。

還有一個更根本的問題：部署不在自己手上。看不到 error log，看不到系統監控數據。出了問題，只能瞎猜。

「那換個工具？」

我試了 Webflow。結果？還是要從 Webflow 101 開始看起。它的排版邏輯，跟 Bubble 完全不同。我在 Bubble 學的東西，一點都帶不過來。

這就是 No-Code 的真相：

**每個工具都是一個封閉世界。** 換一個，從零開始。

## 直到今年 6 月

Claude Code 推出。

我抱著試試看的心態打開它。

然後，就停不下來了。每天 12 小時以上，沉在裡面。

AI 不會頂嘴，不會說「這需求不合理」，不會開會開到天荒地老。你說什麼，它就做什麼。不懂的地方，它解釋給你聽。錯了，它幫你修。

第一週，我戰戰兢兢。

從安裝 VS Code 開始。打開來，黑漆漆的畫面，一大堆程式碼。

「我需要看懂這些嗎？」「還是就讓 AI 處理就好？」

第二週，開始有感覺了。

不是「我會寫程式」那種感覺。是「我知道怎麼跟 AI 協作」的感覺。

因為這就是我在公司裡每天在做的事：定義問題、拆解需求、驗收產出、溝通協調。

只是以前的對象是工程師。現在的對象是 AI。

半年後，三個完整的專案：後端 API、前端 App、Chrome Extension。一個能用的 SaaS，有付費功能、有用戶系統、有 PDF 處理、有 AI 分析。上架 [Chrome Web Store](https://chromewebstore.google.com/detail/ai-resume-advisor-%E2%80%94-linke/iafpieelbhhlmhgjekflneciffoonkgf?fbclid=IwY2xjawPJpIlleHRuA2FlbQIxMABicmlkETFkdGhRQnJtbE1RMUQ2dDJXc3J0YwZhcHBfaWQQMjIyMDM5MTc4ODIwMDg5MgABHhquLa5j6Gc-TvREdWUXHtAPyRpQnfF8Mkz32ymRTS9tSGM8AfrSKuZmrV46_aem_DcQbMik7GTA9nzN0vC_lVQ) 和 [Edge Add-ons](https://microsoftedge.microsoft.com/addons/detail/ai-resume-advisor-%E2%80%94-linke/ekhlndlifglmicdphldpciifohloeaek?fbclid=IwY2xjawPJpJpleHRuA2FlbQIxMABicmlkETFkdGhRQnJtbE1RMUQ2dDJXc3J0YwZhcHBfaWQQMjIyMDM5MTc4ODIwMDg5MgABHp2MkL0o6fOz0aXJMAjeKh82gwYpgRvsABKVpsujR3WbaMLU-m1pLDXWazIr_aem_bqRg4bv8Ay2cQM5dHnu9xQ)。

離開工程師 14 年的人，做出了這些。

## 不一樣的地方

No-Code 解決的問題是：讓不會寫程式的人，也能做產品。Vibe Coding 解決的是同一個問題。

但有一個關鍵差異。

No-Code，你學的是「工具」。Bubble 的邏輯、Webflow 的排版、Zapier 和 n8n 的自動化流程。換一個工具，重新學一次。

Vibe Coding，你學的是「跟 AI 溝通」。今天用 Claude，明天用 GPT，後天用別的。核心能力是一樣的：把問題講清楚、把需求拆開、驗證結果。

這個技能不會因為換工具而歸零。

「可是寫程式要搞伺服器吧？」

以前是。現在不是了。Azure CLI + GitHub Actions，連上 GitHub，自動部署。推上去，就上線了。

## 寫在最後

如果你問我，AI coding 最大的價值是什麼？

不是「寫程式變快」。

是「讓那些想做產品但做不出來的人，終於可以做了」。一個人也可以。

14 年的距離，被一個工具拉近。

有點不可思議。
